Experiment X - Subject/Topic

Description

For this experiment, I wanted to simulate the worlds shown in class. I did this by using noise as a heightmap, then allowing the player to edit the heightmap. The map also periodically breathes as it changes color, giving a calming effect, at least for me.

Technical

To accomplish this, I took the demo code and tweaked the isometric squares to increase in height depending on a fraction of a noise map. I then tweaked the buttonpressed function in the file to affect all neighbouring buttons as well

Reflection

This was fun to do! I have been feeling pretty stressed lately and watching this kind of calms me down.
